筆記：用 Hexo + GitHub page 架設個人網站

發表於 2022-05-18

一直想要架網站放學習新技術的筆記和作品連結，剛好發現了由臺灣人開發的網誌框架 Hexo，可以快速地產生靜態網頁，資訊和插件很豐富，也有相當高的安全性和修改的彈性，對個人網站架設來說是非常方便的工具。
在網誌的空間上我選擇 GitHub page，這是 GitHub 提供的網頁代管服務，可以存放靜態專案。

安裝 git

Git 是版本控制工具，就像遊戲的存檔功能一樣，可以記錄新進度、載入舊進度。我們在本機完成網站後，就要使用 Git 推送到 GitHub 上。
可以在官網下載。

Hexo 是用 Node.js 打造的網誌框架，所以 Node.js 是必備的工具。

SSH-key 設定

我在後面的推送步驟一直遇到 spawn failed，一開始以為是網路或帳號設定有問題，也嘗試過刪掉 _deploy 資料夾、 修改 repo 等方法（若有類似問題可以看Hexo错误：spawn failed的解决方法，裡面介紹了剛剛所說的兩種方法）但皆失敗，最後發現問題可能是出在登入 GitHub 上，於是改成先綁定 SSH 密鑰，就順利解決了，也省下了之後推送要輸入密碼的麻煩。
